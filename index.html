Node.js

  • Node.js is a server-side platform built on JavaScript Engine (V8 Engine). It runs without browser(independed platform) it runing on outside of browser with js runtime environment
  • Node.js is an open source server environment. Node.js allows you to run JavaScript on the server. Node.js uses JavaScript on the server
  • Node.js is an open source, cross-platform runtime environment for developing server-side and networking applications.
  • Advantages :- large community support, high performance, Reusable code, simple to learn, Async programming model, Ability to handle multiple requests.
